# split-text-file
텍스트 파일 분할기 입니다.

✨ 주요 기능
자동 분할: 기본값은 100KB 단위로 텍스트 파일을 나눕니다.
사용자 지정 크기: 원하는 KB 단위로 입력 가능.
GUI 지원: 드래그 앤 드롭, 버튼 클릭으로 파일 선택 가능.
출력 폴더 지정: 분할된 파일을 원하는 위치에 저장.
명령어 실행 지원: GUI 없이 터미널에서도 사용 가능.
🖥️ GUI 화면
프로그램을 실행하면 다음과 같은 창이 뜹니다:


텍스트 파일을 드래그 앤 드롭하거나 버튼으로 선택
저장 위치 지정
원하는 분할 크기 입력 후 "분할 시작" 버튼 클릭
분할이 완료되면 팝업 메시지로 결과를 알려주며, 상태창에도 생성된 파일 개수가 표시됩니다.

🚀 실행 방법
split_text_file.py 더블클릭→ gui 창이 열립니다.

📌 활용 예시
대용량 로그 파일을 분석할 때
머신러닝 데이터셋을 작은 단위로 나눌 때
텍스트 기반 전자책을 챕터별로 분할할 때
📝 마무리
이 프로그램은 단순하지만 실용적입니다. GUI와 CLI를 모두 지원해 초보자부터 개발자까지 누구나 쉽게 사용할 수 있습니다. 필요하다면 기능을 확장해 PDF, CSV 등 다른 파일 형식에도 적용할 수 있겠죠.
